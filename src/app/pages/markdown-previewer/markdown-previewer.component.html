Markdown Guide

This previewer supports GitHub Flavored Markdown (GFM) with the following features:

  • Headers: Use # for H1, ## for H2, ### for H3
  • Bold: **text** or __text__
  • Italic: *text* or _text_
  • Strikethrough: ~~text~~
  • Code: `inline code` or ```language for code blocks
  • Links: [text](url)
  • Images: ![alt](url)
  • Lists: Use -, *, or + for unordered, 1. for ordered
  • Task Lists: - [ ] or - [x]
  • Tables: Use | to separate columns
  • Blockquotes: Use > at the start of lines
